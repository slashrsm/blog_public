Results of the Drupal 8 media sprint

Mon, 02.01.2017 - 17:20
Comments

In December media team met in Berlin and sprinted on Drupal core media improvements for a week.

Today's @drupalmedia sprinters. Awesome group that is making #Drupal the best media management CMS! pic.twitter.com/G2LqD603pH

I demonstrated the results of the sprint in a short video.

Demo is based on the sandbox project that includes the following patches:

  • #2836381-13: Seven's entity-add-list template omits link attributes
  • #2834777-6: Refactor Drupal\system\Plugin\views\field\BulkForm to support a select form as well
  • #2834729-12: Create an MVP for adding and re-using Media
  • #2831940-8: Create file field widget on top of media entity
  • #2831944-4: Implement media type plugin for remote video
  • #2831943-6: Re-create image and file field-like formatters on top of media entity
  • #2831937-23: Add "Image" media type plugin
  • #2831936-40: Add "File" media type plugin
  • #2831274-170: Bring Media entity module to core as Media module
